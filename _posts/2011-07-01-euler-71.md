[By listing the set of reduced proper fractions for d â‰¤ 1,000,000 in ascending order of size, find the numerator of the fraction immediately to the left of 3/7.](http://projecteuler.net/problem=71)

```python
#!/usr/bin/env python

from fractions import gcd

N = 1000000

for d in xrange(2, N):
    for n in xrange(3*d/7+1, 1, -1):
        if n*7 < 3*d:
            if gcd(n,d) == 1:
                res = (n, d)
                break;

print(res)
```
